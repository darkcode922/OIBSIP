# 🚀 PlanNer - Smart Study Ecosystem

PlanNer is a premium, high-end dark-themed landing page designed for a next-generation student study planner. It is engineered with a modern SaaS aesthetic, precise typography, and optimized responsive layouts tailored for high-achieving students.

---

## 🎨 Design & Theme Metrics

- **Primary Color:** `#2563EB` (Electric Blue)
- **Secondary / Background Color:** `#0A0A0F` (Premium Deep Dark)
- **Card/Container Background:** `#12121A` (Card Dark)
- **Accent Color:** `#22C55E` (Accent Green)
- **Typography:** Poppins (Headings) & Inter (Body)

---

## ✨ Features

- **Sticky Glassmorphism Navbar:** High-end blur filter effect with real-time responsive scaling.
- **Engineered for Focus:** Clean semantic blocks showcasing Study Scheduling, Daily Goals, and Progress Analytics.
- **Developer Core Terminal:** Interactive fake visual log command block tailored for tech portfolios.
- **Production-Ready Footer:** Detailed footer layout featuring cloud node simulation and clean social matrix routing.
